# leah-20260609-1

## Vision

A simple database search app. A user can search for an entity, see it resolved, then see a graph of it's relationships with connected entities one hop away. A user can click on one of those other entities to refocus the graph on it. A user can click on the focus entity to see it's properties and property values.
